## laravel-admin 前端组件扩展

    弹出层组件 支持唤出:自定义页 表单修改页
    标签选择器 
    级联标签选择器 
    数据表单控件
    
### 安装
```shell script
composer require dlp/component-js
```
### 发布
```shell script
php artisan vendor:publish --provider="DLP\DLPServiceProvider"
```
